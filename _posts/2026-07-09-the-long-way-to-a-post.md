This is not the way you want to make a post quickly, but the long way now is usually an investment in the future and I hope that's the case here.

If you're reading this then my rudimentary Python + iOS skills are paying off and I have built my post generator.

Sure I'd done this before with Shortcuts somewhat clumsily, but this will in time get better.

I can write the post in Notes, then I can generate a thumbnail in Gemini, and pass it through my Python tool in order to get the Jekyll Front Matter correct, including naming the files (all the things you'd use the "post new" command for, usually), then pass that over to the comprehensive Workjng Copy app to get it up to GitHub Pages.

Still to be worked on would be embedded images and the like. I know about Notes' support there already, but I don't often embed a lot.

But I do like to end with an emoji...
